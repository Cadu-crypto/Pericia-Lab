Passo 1 — Criar projeto no Supabase
1
Acesse supabase.com e crie uma conta gratuita
O plano gratuito suporta até 500 MB de banco e 2 projetos — mais do que suficiente.
2
Crie um novo projeto
Escolha um nome (ex: pericia-lab), defina uma senha forte para o banco e selecione a região South America (São Paulo).
3
Execute o esquema do banco
No painel do Supabase, vá em SQL Editor → New query, cole o conteúdo do arquivo supabase_schema.sql e clique em Run.
4
Copie as credenciais
Vá em Settings → API e copie a Project URL e a anon public key.
